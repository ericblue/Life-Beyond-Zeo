Zeo Raw Data Library

Zeo has released for developers a version of software and a library that enables a user to view their sleep phase and brainwaves in real time using the serial port on the back of Zeo. In addition a Python decoder library is provided that allows users to more easily build applications that use this data. Example source is also provided which allows developers to create their own software to decode Zeo Raw Data. A special version of firmware (2.6.3R) is required to use this library. This library is an ALPHA release and is not supported by Zeo -- please use at your own risk.

Example applications include:

  • A lucid dreaming application that triggers a light or sound when a user enters REM sleep.
  • Tools for hackers or developers interested in Brain Computer Interfaces.
  • Visualize your brainwaves while awake for applications like concentration or relaxation.
  • A separate Web API is also available for those seeking to access Zeo sleep data online, and a Data Decoder Library is available to get offline access to sleep data stored on the SD card.

For documentation, the raw data library, and examples head to our sourceforge page.
